This repository contains code for mask filtering algorithms and slow motion representations of real-time MRI data.
Example datasets can be accessed from this link: https://drive.google.com/drive/folders/1ZIkrpZg2XOQyBb3ppvBOn8xBMW_PIacl?usp=sharing

To use please download the repo and unzip it. 

Please note to save videos you must add a filepath to line 10 of the createSplitScreenVideo.m file.
